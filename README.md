# RockSocial - Backend

## Alumno

Elías Charadía

## Descripción del Proyecto

RockSocial es una aplicación web de tipo red social desarrollada utilizando una arquitectura cliente-servidor. El backend está construido con NestJS y tiene la responsabilidad de gestionar la autenticación de usuarios, la persistencia de datos y la lógica de negocio de la aplicación.

## Tecnologías Utilizadas

### Backend

* NestJS
* TypeScript
* MongoDB
* Mongoose
* Bcrypt
* Multer
* Class Validator
* Class Transformer

### Base de Datos

* MongoDB

### Control de Versiones

* Git
* GitHub

## Sprint 1 - Backend

### Objetivos Cumplidos

* Creación del proyecto backend utilizando NestJS.
* Configuración inicial de la arquitectura modular.
* Integración con MongoDB mediante Mongoose.
* Implementación de validaciones utilizando DTOs y ValidationPipe.
* Implementación de autenticación básica.
* Gestión de almacenamiento de imágenes de perfil.

## Estado Actual

Durante este sprint se desarrolló la infraestructura principal del backend, incluyendo autenticación, persistencia de usuarios y almacenamiento de imágenes de perfil. En los próximos sprints se implementarán las funcionalidades relacionadas con publicaciones, interacción entre usuarios y mecanismos avanzados de seguridad y autorización.
